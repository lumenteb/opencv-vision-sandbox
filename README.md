# OpenCV Vision Sandbox

Small computer vision experiments with Python and OpenCV.

## Current project: Red Object Tracker

This script uses a webcam to detect a red object, find its center, and track its movement on screen.

## Features

- Webcam input
- Red color detection using HSV masks
- Contour detection
- Object center tracking
- Movement trail visualization

## Tech stack

- Python
- OpenCV
- NumPy
